视口

视口分为布局视口、视觉视口、理想视口

视觉视口：用户正在看到的网站的区域

理想视口 ：理想视口，对设备来讲，是最理想的视口 尺寸（想要的）

布局视口 ：一般手机浏览器都默认设置了一个布局 视口，用于解决早期的PC端页面在手机上显示的问题

meta标签

    

通过meta但标签来对视口进行设置

Width属性，用来设置视口的宽度

initial-scale 初识缩放比

maximum-scale 最大缩放比

minimum-scale 最小缩放比

user-scalable 用户是否可以进行缩放

物理像素与物理像素比

物理像素值得是屏幕显示的最小颗粒，pc端开发时1px等于1个物理像素，但移动端不同，例如iPhone8，1px等于2个物理像素，1px能显示的物理像素个数称为物理像素比，所以在iPhone8里物理像素比为2。

早期pc端和手机屏幕物理像素比为1：1，由于视网膜屏技术的出现，将更多的物理像素点压缩到一块屏幕，提高了屏幕分辨率，就有了多倍物理像素。

元素偏移量offset系列

offset 与 style 区别

offset

  • offset 可以得到任意样式表中的样式值

  • offset 系列获得的数值是没有单位的

  • offsetWidth 包含padding+border+width

  • offsetWidth 等属性是只读属性，只能获取不能赋值

  • 所以，我们想要获取元素大小位置，用offset更合适

style

  • style 只能得到行内样式表中的样式值
  • style.width 获得的是带有单位的字符串
  • style.width 获得不包含padding和border 的值
  • style.width 是可读写属性，可以获取也可以赋值
  • 所以，我们想要给元素更改值，则需要用style改变

搭建个人博客的步骤

发表于 2020-06-27 更新于 2021-10-06 分类于 个人博客

1.安装git 官网 https://git-scm.com/download/win 或者 https://npm.taobao.org/mirrors/git-for-windows/（第一个速度太慢，第二个找到最新的版本，安装很快） 用git –version验证版本号

2.安装Node.js 官网 https://nodejs.org/en/download/ 或者 https://npm.taobao.org/mirrors/node 用node–version验证版本号 npm –version

3.npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
cnpm -v #查看cnpm版本

4.cnpm install -g hexo-cli #安装hexo框架 hexo -v #查看hexo版本

—完成以上步骤就可以正式来搭建hexo博客了

5.windows 输入 mkdir C:\blog 看你要放在哪一个盘符，这样就创建了一个blog目录
